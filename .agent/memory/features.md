# Features List

## 1. Student Profiles Management
- Creation of student profiles (name, batch, place, month of joining, unique student ID, password).
- Editing and deleting student records.
- Batch filter management for Repeater, Weekend Batch, Crash Course, and Test Batch.

## 2. Marks Entry Systems
- **Weekly Tests**: Marks entry for Biology, Chemistry, and Physics weekly tests. Calculates score and unanswered questions automatically from total correct and wrong inputs.
- **Grand Monthly Test**: spreadsheet-style, single-screen grid editor. Auto-calculates total correct, total wrong, unanswered questions, and final marks based on 6 inputs (Biology C/W, Physics C/W, Chemistry C/W). Allows negative scores.

## 3. Data Integration & Export
- Save test records containing student scores directly to Supabase.
- Excel export of current test grid displaying student names, correct/wrong subject scores, totals, unanswered, and final marks.

## 4. Student Portal
- Secure student login.
- Performance charts displaying score trends.
- Grand Test history with expandable subject-wise detail drawers showing parents/students how final marks were achieved.

## 5. Admin Student Analytics
- Trend analysis lines graphs for each subject.
- History registers with inline expandable detail drawers for Grand Tests.
- Subject-wise average correct, wrong, and score cards.
