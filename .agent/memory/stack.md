# Technology Stack

## Core Frameworks
- **Framework**: Next.js 15.5.19 (App Router)
- **Runtime**: React 19.1.0 / Node.js
- **Language**: TypeScript 5.x
- **Styling**: Tailwind CSS v4.0

## Database & Backend services
- **Backend as a Service**: Supabase (PostgreSQL database, Auth, Session Management)
- **Libraries**: `@supabase/supabase-js`, `@supabase/ssr`

## Utility Libraries
- **Charts**: Recharts (Custom SVG charts are also used in dashboard displays)
- **Animation**: Framer Motion
- **Cryptography**: `bcryptjs` (for local verification of credentials if needed)
- **Data Export**: `xlsx` (sheetjs) for exporting student marks spreadsheet data
- **Icons**: Lucide React
