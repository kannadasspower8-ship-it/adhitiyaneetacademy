# Build Business Website

Objective

Create a production-ready business website.

Workflow

1. Read orchestrator.md.
2. Read project memory.
3. Read project rules.
4. Project Manager creates a development plan.
5. Architect designs the structure.
6. UI Designer creates the interface.
7. Frontend Engineer builds reusable components.
8. Backend Engineer builds backend if required.
9. SEO Engineer optimizes every page.
10. QA Engineer tests the project.
11. Deployment Engineer prepares production.
12. Update memory.
13. Generate summary.
